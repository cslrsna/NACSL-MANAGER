*Wordpress extension*
# NACSL-MANAGER
***

This extension add some custom feature to simplify the webmaster job of a CSL with managing his WordPress web site.

#### Custom post type:
- meeting
- service subcommittee
- activity

#### Custom taxonomies
- type of meeting
  - shortcut
- city of the meeting
- day's of meeting
